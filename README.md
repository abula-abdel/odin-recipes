# odin-recipes
Repo for the Odin recipes project

In this project, I will create a basic recipies website. This will include links to recipes I like or find interesting. 

Skills Demonstrated:

- Create and clone Github repo on to local machine
- Create html files with boilerplate code
- Created an index.html file to serve as the main webpage
- Linked each html file back to orignal through href. These are put in an ordered list
- Added images to each html file
- Used different headings to segment the webpage. Ordered and unordered lists used to make instructions and ingredients needed
- Created nested lists and elements found within the instructions
